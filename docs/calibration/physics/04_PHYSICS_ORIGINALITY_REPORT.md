# Physics Originality Report

Confirms every candidate question is **independently authored**. Official MATSEC papers were used only to understand *abstract examination demand* (which syllabus outcomes are assessed, command words, typical marks, reasoning steps, common misconceptions, broad difficulty). **No** official wording, values, diagrams, scenarios, distractor sets or reasoning sequences were copied or lightly paraphrased. There is deliberately **no one-to-one "official question → KPC rewrite" mapping.**

## What was independently created (all 40)
- **Wording:** every prompt written from scratch in KPC's plain style.
- **Context:** everyday/Maltese-flavoured scenarios chosen independently (tram, crate, swimmer, immersion heater, local radio at 150 MHz, Sun–Mars, etc.).
- **Values:** all numbers chosen by the author to give clean, checkable answers; they differ from both the previous KPC bank and typical paper values.
- **Response structure:** single numeric answer or 4-option MCQ; option order and distractors authored to reflect named misconceptions.
- **Worked solutions:** written independently, step-by-step.
- **Diagrams:** only the pre-existing **original KPC SVGs** (`vtgraph`, `wave`, `circuit`) are reused, with question numbers chosen to match them; no official diagram is reproduced.

## Abstract demand analysed (examples, not sources)
- LO2: recognise that resultant force drives acceleration; convert km/h→m/s; multi-step braking. (Demand pattern only.)
- LO1: refraction/refractive index, TIR, magnification — standard Level-2/3 optics demand.
- LO5: transformer turns/current relationships. LO6: half-life counting. LO7: standard-form astronomical distances.

## Independence rationale
Because the abstract skills (e.g. "compute acceleration from a resultant force") are syllabus-defined and shared by all exam boards, testing the same *skill* is expected and legitimate; what must be original is the *expression* — wording, numbers, context, distractors, solution — all of which are freshly authored here. Any residual resemblance is at the level of universal physics facts (e.g. "alpha is stopped by paper"), which are not copyrightable.
