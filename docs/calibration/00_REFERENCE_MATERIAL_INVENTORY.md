# Reference Material Inventory — Physics & Mathematics

**Prepared:** 2026-07-12 · **Branch:** `diagnostic-calibration` · **Status:** document-readiness check (no calibration performed yet)

This inventory records the official MATSEC/SEC reference material supplied under the
**git-ignored** `reference/` folder. Every document below was identified from its
**own printed content** (cover text, exam code, dates) via image inspection and text
extraction — not from filenames alone. Files were then renamed to a consistent scheme
(mapping in Appendix 1). The reference PDFs themselves remain private and are never
committed to Git.

**Key facts established during inspection**
- Subject codes: **SEC 24 = Physics**, **SEC 23 = Mathematics**.
- The exam-code suffix encodes year + session (e.g. `.24m` = 2024 Main/May, `.23s` = 2023 September).
- **Exam-structure change from 2025:** 2025 papers carry the new code `…/1L1-2-3` ("Level 1-2-3"); 2021–2024 papers use the older `…/1`, `…/1a`, `…/1b` codes.
- Current syllabi are present for **both** subjects: **Physics** (SEC 24, 2026 edition, updated April 2024) and **Mathematics** (SEC 23, 2026 edition). *(The Maths syllabus was added on 2026-07-12, after the first draft of this inventory.)*
- Marking schemes and examiners' reports exist for **Main (May) sessions only** — this is normal MATSEC practice; September resit papers are not accompanied by published schemes/reports.

## D. Syllabus compatibility

### Physics
- **Current syllabus available:** SEC 24 (2026), updated April 2024 — governs the current cohort.
- The **2025 papers already use the new "Level 1-2-3" structure** that matches this syllabus generation, so **2025 papers are the best structural match** to the current syllabus.
- The **2021–2024 papers use the older format** (`SEC24/1`). Treat them as **relevant-with-caution**: core O-level physics is stable, so they remain useful for difficulty and command-word analysis, but paper structure and possibly some content emphasis differ. Confirm specifics against the 2026 syllabus during the audit (not done here).
- Both available marking schemes (2021, 2024) are **old-format**; useful for marking conventions but not for new-format structure.

### Mathematics
- **Current syllabus now available:** SEC 23 (2026) — governs the current cohort, same generation as the Physics 2026 syllabus.
- The **2025 papers use the new "Level 1-2-3" format** matching this syllabus generation → **2025 papers are the best structural match**; 2021–2024 are older format (relevant-with-caution, as for Physics).
- Mathematics is now the **best-equipped subject**: current syllabus + full 2021–2025 paper run + three recent marking schemes (2023, 2024, 2025 — including the new-format 2025) + five examiners' reports. Coverage mapping is no longer blocked.

*Per the brief, the detailed "topics added/removed/changed" comparison is deferred to the academic blueprint stage and is not attempted in this readiness check. The 2025 code-format change is the signal to investigate there.*

## F. Copyright and originality reminder

Official MATSEC/SEC materials in `reference/` may be used **only to analyse abstract
examination demand**, specifically:
- syllabus coverage and topic weighting;
- command words and question phrasing conventions;
- cognitive demand and reasoning steps expected;
- typical mark allocations and expected answer precision;
- common misconceptions noted by examiners;
- broad difficulty and discrimination.

Official **wording, diagrams, numerical values, scenarios/contexts, answer options
(distractors) and question structures must NOT be copied or lightly paraphrased** into
the KPC production question banks or any proposed questions. KPC diagnostic questions
must remain **original**, written in exam style but independently constructed. These
reference PDFs must never be committed to Git, deployed, or redistributed.
